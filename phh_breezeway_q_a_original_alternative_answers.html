Q5 Can the system send text messages or link to Slack channels? Breezeway supports in‑app, email, and SMS for staff. Guest messaging is a paid add‑on. Slack linking not supported; use an email‑to‑Slack bridge. Keep Breezeway as the single comms hub: use staff push/email/SMS for ops, and if Slack visibility is needed, forward a designated ops inbox into Slack so nothing is double‑managed. Standardize who gets what (e.g., cleaners = push; supervisors = push+email) in a brief Comms SOP.
Q8 How are housekeepers notified of new cleans? App / Email / SMS? Housekeepers choose push/email/SMS. Save the mobile number to enable SMS; staff SMS is included. Beyond preferences, ensure tasks are Committed (not Draft) and the cleaner is the assigned/default worker so notifications fire. Use SMS for same‑day turns; default to push to reduce inbox noise.
Q10 Do current maintenance tasks auto‑import or bulk import? No auto‑import or bulk import; add new tasks manually. Stand up recurring maintenance workflows (by asset/property) so the next 60 days of work schedules itself from rules. Manually backfill only truly “in‑progress” legacy items to reach automation faster.
Q12 Is there a way to auto‑import photos? No. Staff must capture verification photos in the Breezeway mobile app. Keep verification live, but add Reference Photos to the template (e.g., link to a property album) so reps can see standards, then require new photos on completion via “Photo required” checklist items.
Q16 Add specific tasks to a departure clean based on property tag? Not by tag. Use Template Modifiers or dedicated templates (e.g., Spa, Pet‑Friendly). Mirror “tags” using property groups and attach a small automation that adds an “add‑on” task when a checkout occurs at any property in that group. Keeps templates simpler while preserving automation.
Q19 Can we change Department Names? Not currently. On the roadmap; timing not guaranteed. Simulate departments with Task Types + Saved Views (e.g., “Grounds – Lawn,” “Grounds – Pool”) and adopt a naming convention in task titles: Property – TaskType – Date for clear filters and exports.
Q22 Can we change colours in the property schedule? No colour customisation; log an enhancement request. Reduce colour dependency: save filtered Schedule views (by department/assignee) and rely on Property Readiness (green check) to confirm guest‑ready status at a glance.
Q25 Best way to schedule lawn mowing (every 2 weeks Oct–Apr; every 4 weeks May–Sept)? Two seasonal workflows or date‑bounded recurrence. Use two rules with fixed weekdays (e.g., every 2nd Monday vs every 4th Monday) bounded by season dates. Add a “Skip if heavy rain” checklist note and move the single occurrence forward one day when needed to keep history intact.
Q27 HomHero auto‑emails housekeepers—how will this work with Breezeway? Pick one source of truth; many disable HomHero emails and rely on Breezeway notifications. Document your approach. Segmented approach: keep HomHero emails on for external vendors, and rely on Breezeway push/SMS for internal staff. Add a one‑page vendor onboarding sheet (“Which app tells you what”) to avoid mixed instructions.
Q28 How do housekeepers report maintenance & where do we view it? Reps tap Add Issue in the mobile app; managers are notified per settings; issues appear in the Maintenance tab. Bake it into the cleaning template: add a required item “Any issues to report?” If “Yes,” a dependent automation creates a Maintenance task and pings a Supervisor for triage—so issues are captured even if someone forgets Add Issue.
Q30 Auto‑allocate jobs > $330 to owner? Where do we input the $$? Not supported. Workaround: assign owner as Representative and assign the task to them. Use a lightweight approval workflow: add a checklist question “Estimate > $330?” (required). If “Yes,” auto‑create an Owner Approval task (or notify the owner) and tag the original job “Awaiting Owner Approval.” Proceed once approved.
Q21 Can we modify or hide guest ratings? No. Guest ratings feed property scores; cleaner departure ratings are internal‑only (task reports). Don’t collect public ratings by default: disable any rating request messages in guest campaigns, and gather private CSAT via a short internal survey or owner‑safe follow‑up. Preserves scoring while controlling how/when feedback is requested.
